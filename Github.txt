Github

1. Pengertian
GitHub merupakan manajemen project, sistem versioning code, sekaligus platform jaringan sosial bagi para developer seluruh dunia.
Banyak sekali fungsi yang bisa digunakan oleh para developer, sehingga melalui platform ini memudahkan developer dalam mengembangkan sebuah karya.
2. Cara Menggunakan Github bagi pemula
- Join Account
- Verifikasi Alamat E-mail
- Lengkapi Akun Profil
- Start a Project
- Create a New File
- Bersosialisasi
